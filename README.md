# Authentication System (Next.js + Golang)

This project demonstrates a basic authentication flow with a frontend and backend setup.

## Tech Stack
- Frontend: Next.js
- Backend: Golang

## Features
- Signup using email or mobile number with password
- Login using email or mobile number with password
- Password hashing using bcrypt

## Notes
- This was my first hands-on task using this tech stack
- The focus was on understanding and implementing the end-to-end flow
- User data is stored in-memory for demo purposes
